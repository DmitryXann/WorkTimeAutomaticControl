Description:

Work Time Automatic Control StopWatch use Google Spreadsheets to save your work information in the cloud.

Default spreadsheet name: WorkReport.

If for some reason saving information fails, C# program automatically saves all information in local file. When you start again C# program, it will automatically try to save local saves in the cloud, if saving was successful, program clears local file.
Local file structure:

<?xml version=\"1.0\" encoding=\"utf-8\"?>

<WorkTimeAutomaticControlCrashReport>

<!--This is auto generated file, do not modify-->

    <WorkTime WorkTime=\"YYYY-MM-DDTHH:MM:SS\">Work description</WorkTime>

</WorkTimeAutomaticControlCrashReport>"
